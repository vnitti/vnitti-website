    About

    PPL Workout Generator is a small app that will help you to generate daily randomized workouts following the Pull-Push-Leg principle. Also, it will guide you through your workout so you don't lose count on the exercises, rest times and the total duration.

    This app only includes exercises that can be done with body weight, dumbells, elastic bands or exercises that can be done with machinery that can be substituted with any of the previous elements. Exercises that are meant to be done only with gym machinery are not included.

    Developed by Vicente Nitti using HTML, CSS and JavaScript. Check his website here.


    Instructions

    1. Select the type of exercise you want
    2. Select the number of exercises and sets
    3. Specify the rest times between exercises and sets
    4. When you are set, press the "Generate Workout" button
    5. Press "Start" to start your workout
    6. You can skip to the next element of the list, pause, resume or quit your workout with the panel button
